🚀 CÓMO SUBIR A GITHUB (3 OPCIONES)
═══════════════════════════════════════════════════════════════════════════

    ┌─────────────────────────────────────────────────────────────────────┐
    │ OPCIÓN 1: Script Automático (MÁS FÁCIL) ⭐                         │
    └─────────────────────────────────────────────────────────────────────┘

    1. Asegúrate de tener Git instalado:
       https://git-scm.com/download/win

    2. Ejecuta:
       subir_github.bat

    3. Cuando te pida credenciales:
       Usuario: rrevelesm
       Contraseña: Tu Personal Access Token


    ┌─────────────────────────────────────────────────────────────────────┐
    │ OPCIÓN 2: Comandos Manuales                                         │
    └─────────────────────────────────────────────────────────────────────┘

    git init
    git add .
    git commit -m "🎓 Primer commit: Sistema UPIIZ"
    git branch -M main
    git remote add origin https://github.com/rrevelesm/DeptoInv_UPIIZ.git
    git push -u origin main


    ┌─────────────────────────────────────────────────────────────────────┐
    │ OPCIÓN 3: GitHub Desktop (Interfaz Gráfica)                         │
    └─────────────────────────────────────────────────────────────────────┘

    1. Descarga GitHub Desktop:
       https://desktop.github.com/

    2. Inicia sesión con tu cuenta (rrevelesm)

    3. File → Add Local Repository → Selecciona tu carpeta

    4. Commit y Push


═══════════════════════════════════════════════════════════════════════════
🔑 CREAR PERSONAL ACCESS TOKEN
═══════════════════════════════════════════════════════════════════════════

    Si Git te pide contraseña y no funciona:

    1. Ve a: https://github.com/settings/tokens

    2. "Generate new token" → "Generate new token (classic)"

    3. Configuración:
       - Nota: "Token para DeptoInv_UPIIZ"
       - Expiration: 90 días
       - Permisos: ✅ repo (marcar todo)

    4. Generate token

    5. COPIA EL TOKEN (ghp_xxxxxxxxxxxx)

    6. Úsalo como contraseña cuando hagas git push
