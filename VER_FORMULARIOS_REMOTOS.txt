  OPCIÓN A: Con túnel SSH (RECOMENDADO)
  ──────────────────────────────────────

    1. En PowerShell, ejecuta:
    
       ssh -L 5000:localhost:5000 tu-usuario@tu-servidor
    
    
    2. Dentro del servidor, ejecuta:
    
       cd ~/DeptoInv && python app.py
    
    
    3. Abre tu navegador en:
    
       http://localhost:5000/admin-login
    
    
    4. Ingresa el token:
    
       bRIWynRiD-kz0KsyHtrgvuyEbQtBgNXBaGqWlOX1cIU


  OPCIÓN B: Acceso directo por IP (si el firewall lo permite)
  ────────────────────────────────────────────────────────────

    1. Conectarte al servidor:
    
       ssh tu-usuario@tu-servidor
    
    
    2. Iniciar el servidor:
    
       cd ~/DeptoInv && python app.py
    
    
    3. Abrir en navegador (desde tu PC):
    
       http://IP-DEL-SERVIDOR:5000/admin-login
    
       Ejemplo: http://192.168.1.100:5000/admin-login
    
    
    4. Ingresar token:
    
       bRIWynRiD-kz0KsyHtrgvuyEbQtBgNXBaGqWlOX1cIU
