  ❌ El servidor no inicia:
     - Verifica que Python esté instalado: python --version
     - Reinstala dependencias: pip install -r requirements.txt

  ❌ No puedo acceder al panel:
     - Verifica que el servidor esté corriendo
     - Abre: http://localhost:5000/admin-login
     - Ingresa el token de arriba

  ❌ Olvidé el token:
     - Ejecuta: ACCESO_RAPIDO.bat → Opción [4]
     - O abre: CREDENCIALES_ADMIN.txt

  ❌ El puerto 5000 está ocupado:
     - Busca el proceso: netstat -ano | findstr :5000
     - Cierra el proceso anterior
