📋 PASO 1: EN LA CONSOLA BASH DE PYTHONANYWHERE
═══════════════════════════════════════════════════════════════════════════

    Menú: Consoles → Bash


    ┌─────────────────────────────────────────────────────────────────────┐
    │ COMANDO 1: Clonar repositorio                                       │
    └─────────────────────────────────────────────────────────────────────┘

    git clone https://github.com/rrevelesm/DeptoInv.git


    ┌─────────────────────────────────────────────────────────────────────┐
    │ COMANDO 2: Entrar al directorio                                     │
    └─────────────────────────────────────────────────────────────────────┘

    cd DeptoInv


    ┌─────────────────────────────────────────────────────────────────────┐
    │ COMANDO 3: Ver archivos                                             │
    └─────────────────────────────────────────────────────────────────────┘

    ls -la


    ┌─────────────────────────────────────────────────────────────────────┐
    │ COMANDO 4: Instalar dependencias                                    │
    └─────────────────────────────────────────────────────────────────────┘

    pip3.10 install --user flask flask-cors sqlalchemy pandas openpyxl werkzeug


    ┌─────────────────────────────────────────────────────────────────────┐
    │ COMANDO 5: Inicializar base de datos                                │
    └─────────────────────────────────────────────────────────────────────┘

    python3.10 database.py


    ┌─────────────────────────────────────────────────────────────────────┐
    │ COMANDO 6: Importar proyectos aprobados                             │
    └─────────────────────────────────────────────────────────────────────┘

    python3.10 scripts/importar_solo_aprobados.py

⚙️ PASO 3: CONFIGURAR WSGI
═══════════════════════════════════════════════════════════════════════════

    En pestaña Web → Sección "Code" → Clic en:
    /var/www/upiizinvestigacion_pythonanywhere_com_wsgi.py


    ┌─────────────────────────────────────────────────────────────────────┐
    │ BORRAR TODO Y PEGAR ESTE CÓDIGO:                                    │
    └─────────────────────────────────────────────────────────────────────┘

import sys
import os

# Configurar path del proyecto
path = '/home/upiizinvestigacion/DeptoInv'
if path not in sys.path:
    sys.path.insert(0, path)

# Configurar directorio de trabajo
os.chdir(path)

# Importar la aplicación Flask
from app import app as application

# Configuración para producción
application.config['DEBUG'] = False


    Guardar: Clic en "Save" (botón verde)

🆘 SI HAY ERRORES
═══════════════════════════════════════════════════════════════════════════

    Pestaña Web → Log files → Error log

    Copia el error y dímelo.


    Errores Comunes:
    ────────────────

    1. "ImportError: No module named flask"
       → Instalar de nuevo: pip3.10 install --user flask flask-cors

    2. "No such file or directory"
       → Verificar path en WSGI: /home/upiizinvestigacion/DeptoInv

    3. Página en blanco
       → Ver Error log
       → Inicializar BD: python3.10 database.py
